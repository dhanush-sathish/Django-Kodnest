# Django-Kodnest

A Django-based project for the Kodnest platform.

## Getting Started

### Prerequisites

- Python 3.8+
- Django

### Installation

```bash
git clone https://github.com/dhanush-sathish/Django-Kodnest.git
cd Django-Kodnest
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

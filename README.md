# Django-Kodnest

A Django-based application for the Kodnest platform.

## Overview

Django-Kodnest is a web application built with the Django framework, designed to support the Kodnest learning and coding platform.

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/dhanush-sathish/Django-Kodnest.git
cd Django-Kodnest
pip install -r requirements.txt
```

### Running the Development Server

```bash
python manage.py migrate
python manage.py runserver
```

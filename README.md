# Django-Kodnest

A Django-based application for the Kodnest platform.

## Overview

Django-Kodnest is a web application built with Django, designed to support the Kodnest learning platform.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Apply migrations:
   ```bash
   python manage.py migrate
   ```

3. Run the development server:
   ```bash
   python manage.py runserver
   ```

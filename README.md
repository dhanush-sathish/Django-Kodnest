# Django-Kodnest

A Django-based application for the Kodnest platform.

## Overview

Django-Kodnest is a web application built with Django, designed to support the Kodnest learning platform.

## Getting Started

### Prerequisites

- Python 3.8+
- Django 4.x

### Installation

```bash
git clone https://github.com/dhanush-sathish/Django-Kodnest.git
cd Django-Kodnest
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
